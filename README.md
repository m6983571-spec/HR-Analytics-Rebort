# HR-Analytics-Rebort
This interactive sales dashboard provides a complete comparison of sales performance across retailers, cities, products, and sales channels.

It allows users to analyze which retailer achieved the highest sales, which city generated the strongest performance, and which product categories were the best sellers. The dashboard also compares different sales methods such as in-store, online, and outlet to determine the most effective sales channel.

Through clear visualizations and filters, users can easily explore trends, compare dealers, evaluate product performance, and identify top-performing markets.

The main objective of this dashboard is to support data-driven decision-making by highlighting sales distribution, product demand, retailer performance, and city-level results in an interactive and user-friendly way.
